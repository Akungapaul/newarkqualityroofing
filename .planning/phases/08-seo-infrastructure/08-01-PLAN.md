     - `comparisons`: all comparisons mapped to `${BASE_URL}/${comparison.slug}`
     - `articles`: all articles mapped to `${BASE_URL}/${article.slug}`
   - Skip priority and changefreq (Google ignores them per user decision)
   - Import data from existing data modules (services, cities, combos, comparisons, articles, corePages)

   - NOTE: Using `sitemap-index.xml` path to avoid conflict with `sitemap.ts` (which claims `/sitemap/[id].xml`)
